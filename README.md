
# GameHub

Mern stack E-commerce application.

### Install dependencies

✔️ Server :

npm install

✔️ Client :

cd client

npm install

### Run Server

npm run server 

### Run Client

cd client

npm start

### Run concurrently (Both Client and Server)

npm run dev
